小程序01

Posted on 2020-01-01 Edited on 2020-11-19

今天总结

小程序与 PC 开发的区别（⭐⭐）

核心的区别 运行环境不同
pc 运行环境在浏览器
小程序 运行环境在微信
环境不同导致 api 不一致

如何创建一个小程序（⭐⭐⭐）

  1. 注册公众号平台的账户 （年满 18 岁）
  2. 需要拿到 appID 后台 -> 开发 -> 开发设置 -> appID
  3. 不要勾选 云开发

小程序的包结构（⭐⭐⭐）

components 存放组件的
pages 目录： 存放页面的
utils 目录： 存放工具 js 文件的
app.js 小程序的入口文件
app.json 小程序全局配置文件
① pages 存放的是页面的路径 相当于是我们 vue 中的路由
② window 存放的是窗体相关的属性， 例如 navigationBar
app.wxss 小程序全局的样式文件
project.config.json 项目的配置项
sitemap.json 是否被索引 类似于 PC 的 SEO

小程序页面的组成（⭐⭐⭐）

由 4 部分构成

  • .js 文件 小程序页面的逻辑文件（生命周期，事件）
  • .wxml 文件 小程序页面的结构模板 类似于 html 的语法，但是有自己的标签
  • .wxss 文件 小程序页面的样式文件 自己的单位 rpx 专门用来解决移动端适配问题 等分成 750 等分
  • .json 文件 小程序页面的配置文件 （页面的 navigationBar 的样式，或者是注册自定义组件）

宿主环境

了解到什么是宿主环境？ （⭐⭐⭐）
寄生环境 依赖环境

应用程序能够运行的环境
小程序能够运行的环境？ 必须要有微信的 app

通讯（⭐⭐）

通讯主体：
渲染层： wxml 和 wxss
逻辑层： js 代码 xws 脚本语言

运行机制（⭐⭐）

小程序里面的交互都是要依托于微信这个 app 作为中转的
不管是 页面跟 js 交互
或者是 js 调用了第三方服务 api

组件（⭐⭐⭐）

view 组件类似于 div
text 组件 类似于 span
swiper 轮播图 必须要配合 swiper-item 属性 参考文档
scroll-view 滚动组件
① 内容要超过组件宽高
② 设置 scroll-y 属性 或者是 scroll-x 属性
rich-text 解析 html 结构
image 图片组件 mode 设置图片展示的模式 aspectFit
button 组件 按钮
…

api（⭐⭐）

小程序 api 的分类

  • 事件
  • 同步
  • 异步

协同开发（⭐⭐）

管理者两个角色
① 项目开发人员
② 项目体验人员

小程序的启动流程 页面的 启动流程（⭐）
小程序启动流程：
① 下载小程序包
② 解析 app.json 文件找到 pages 里面首页
③ 执行入口文件 app.js 调用 App() 得到实例
④ 渲染内容

页面的启动流程：
① 解析 页面的 json 文件
② 执行对应 .js 文件 Page() 得到实例
③ 找到对应页面进行渲染
