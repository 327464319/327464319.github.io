小程序-Day04 总结

wxs 外联方式（⭐⭐）

  1. 创建一个文件，后缀名是 .wxs, 通过 module.exports 对外进行函数或者是遍历的导出

  2. 在需要使用这个 wxs 的 wxml 页面中，通过 wxs 标签进行引入

外链的方式用的比较多，复用性更强

自定义组件

基本使用（⭐⭐⭐）

  1. 创建自定义组件
    ① 根目录创建一个 components 文件夹
    ② 借助小程序工具，右键创建 component
  2. 在页面中引入自定义组件
    ① 全局引入： 在 app.json 文件中，跟 window 节点同级，声明一个 usingComponents 节点，通过 key value 的形式配置，key 就代表使用的标签名，value 就是自定义组件的路径
    ② 局部引入： 在对应页面的 json 文件中，声明一个 usingComponents 节点，通过 key value 的形式配置，key 就代表使用的标签名，value 就是自定义组件的路径
