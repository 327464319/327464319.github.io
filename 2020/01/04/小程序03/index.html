小程序-Day03 总结

网络请求（⭐⭐⭐）

小程序中对网络请求校验是非常严格的，请求的协议必须是 https 协议
域名需要在后台进行配置（合法域名）
在开发期间可能线上服务器还没有弄好，此时是测试服务器，我们可以通过微信开发工具，详情 -> 本地设置 -> 勾选上不校验合法域名； 项目上线还是必须要在 后台进行配置

编程式导航（⭐⭐⭐）

导航传参（⭐⭐⭐）

  1. 拼接字符串，queryString 的格式，在 url 地址后面 加上 ?key=value&key1=value1

  2. 目标页面的 onLoad 函数里面就可以接受到参数， 通过 这个函数的 形参

下拉刷新&上拉加载（⭐⭐⭐）

下拉刷新

  1. 在对应页面的 json 文件 开启下拉刷新 enablePullDownRefresh: true

  2. 需要在对应的 js 里面监听下拉刷新的函数 onPullDownRefresh()

  3. 需要在这个函数里面，我们手动调用 wx.stopPullDownRefresh() 关闭下拉刷新效果

上拉加载

  1. 在对应页面 的 json 文件， 配置触底距离（可以不配置，默认是 50px） onReachBottomDistance: 100

  2. 只要触发了触底，就会调用 js 里面 onReachBottom() 这个函数，我们就可以去做分页加载逻辑了
